1. Подключите ваше устройство к USB порту
2. Запустите приложение A-Explorer.exe
3. Выберите COM порт, к которому подключено ваше устройство. Если не обнаружено, то нажмите кнопку Scan
4. Подключитесь к устройству нажав кнопку Connect

Использование приложения:

Upload - загрузите файл в устройство
Download - скачайте файл из устройства
Remove - удалите файл
Reboot - перезагрузите устройство
Format - форматирование памяти устройства (ВНИМАНИЕ! Все файлы внутри устройства будут удалены. Прошивка не затрагивается)

В окне справа список файлов на вашем устройстве


1. Connect your device to the USB port
2. Launch the application A-Explorer.exe
3. Select the COM port to which your device is connected. If not detected, then press the Scan button
4. Connect to the device by pressing the Connect button

Using the app:

Upload - upload the file to the device
Download - download the file from the device
Remove - delete the file
Reboot - restart the device
Format - formatting the device memory (ATTENTION! All files inside the device will be deleted. Firmware is not affected)

In the window on the right there is a list of files on your device
