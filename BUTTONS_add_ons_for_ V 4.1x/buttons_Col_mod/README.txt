Набор Рождественских кнопок для прошивки H-RADIO 4.1x
02.12.2022 http://harduino.ru

Поместите файлы с расширением HEX в память приемника с помощью файлового менеджера A-Explorer
Для возврата к стандартным кнопкам удалите файлы из приемника.

Set of Christmas buttons for H-RADIO 4.1x firmware
02.12.2022 http://harduino.ru

Place files with the HEX extension in the receiver's memory using the A-Explorer file manager
To return to the standard buttons, delete the files from the receiver.
