  Additional SPIFFS for V 4.15  V 4.16

Set of  Color_modif buttons/ Black_&_white buttons/ Christmas buttons for H-RADIO 4.1x firmware
02.12.2022 http://harduino.ru


For Color_modif buttons/ Black_&_white buttons .....Modify, compile and save file Button.ini ,
 place this file and all files with the HEX extension in the receiver's memory
 using the A-Explorer file manager.
( for Christams buttons  upload only all .hex  files ).

To return to the standard buttons, delete the files from the receiver.
